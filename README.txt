below are the instructions on how to run the program

1. First, the server.py needs to needs to be ran. It will ask for port number and IP, Once both the port and Ip is given
2. Run client.py and re-enter the same Ip adress and the port number.
3. You can then enter any desired command here
3. First we can type help to check the following commands
4. Then, we can use put command to transfer file from client directory to server directory. For ex: put photo.jpg
5. Then we can use get command to transfer file from server directory to client directory. For ex: get hello.txt
6. we can change the name of the file in server using change command. For ex: change hello.txt newhello.txt
7. In the end we can use bye command to end the program. For ex: bye
